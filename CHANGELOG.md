# Changelog

All notable changes to TaskLane will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial release
- Windows 11 style taskbar
- Window thumbnails on hover
- Focus individual windows (with Accessibility permission)
- Multi-monitor support
- Space/Desktop filtering - shows only current desktop windows
- Pinned apps with drag & drop reordering
- System tray with clock and date
- Settings window with customization options
  - Position (bottom, top, left, right)
  - Size adjustment
  - Blur effect toggle
  - Center icons option
  - Show/hide date in clock
- Menu bar icon with quick access
- French and English localization

## [0.1.0] - 2025-02-05

### Added
- First public release
- Core taskbar functionality
- Basic window management
- Settings UI
